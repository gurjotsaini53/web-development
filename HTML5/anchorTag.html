href --- Attributes help to provide the links to normal text and this provide the destination .

target Attributes

  1. _blank: Load the URL to the new tab on its own .
  2. _self : load the URL to the current tab or browser window. This is default setting .
  3. _parent : Load URL in the context of the parent browsing . It's the same thing as itself if there are no parents.
  4. _top : Load the URL in the high level context of browsing . It is the same thing if there is no parent
